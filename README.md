# sails_01

a [Sails](http://sailsjs.org) application

## 1. Install sails and setup
```shell
$ npm install sails -g
$ sails new sails_01
$ cd sails_01
$ sails lift
```
## 2. Install bower
```shell
  $ echo '{"directory" : "assets/vendor"}' > .bowerrc
  $ npm install bower -g

  $ bower init
  $ bower install bootstrap --save
  $ bower install angular --save
  $ bower install angular-ui --save
```
